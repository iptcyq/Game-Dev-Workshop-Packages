# Game Dev Workshop Packages
Files used for Marvel Club @ EEE 's game dev workshops. Includes templates for participants, and package of the final, complete goal.

| Workshop Name | Date | What we did | Goal |
| -------- | ------- | ------- | ------- | 
| Godot Workshop| 17/10/2025 | Recreated Asteroids | Teach basic Godot |
| Halloween Unity Workshop | 31/10/2025 | Recreated (a significantly worse version of) Silksong | Making 2D platformers in Unity |
| Mobile Game | 30/01/2026 | Recreated Among Us | How to make a mobile game |
| Local Multiplayer Workshop | 10/02/2026 | Generic 2D puzzle platformer inspired by Fireboy and Watergirl | How to make a local multiplayer game |


**Note:** 
- A lot of these are (poorly made) remakes of existing games, with the games' assets. 
- Credit to the sources should mostly be in the individual projects.
- These are for educational purposes
